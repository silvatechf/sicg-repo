## Cómo añadir un nuevo ADR

1. Copiar el formato de un ADR existente (Estado / Contexto / Decisión / Consecuencias)
2. Numerar secuencialmente: `000N-titulo-corto.md`
3. Añadir la fila a la tabla de arriba
